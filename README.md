# poly-polymarket-agent

Deployment and workspace scaffolding for running `openclaw/openclaw` as a Slack-controlled orchestration agent on AWS Lightsail.

Current focus:
- Lightsail setup on the `$5/month` plan.
- OpenAI model via `OPENAI_MODEL` (GPT 5.2-ready).
- Slack Socket Mode control channel.
- Baseline ops/security and spend guardrails.
- Multi-app orchestration scaffold for future Polymarket adapters.
